Conceptos fundamentales
CSS3

¿Qué es CSS3?

CSS significa Cascade Style Sheets, también llamado Hojas de Estilo en Cascada. CSS es un lenguaje de marcado que se emplea para dar formato a un sitio web. Es decir, funciona en conjunto con los archivos HTML. Por esta razón, para crear un sitio web debes saber tanto HTML como CSS.


¿Para qué nos sirve CSS?

Las Hojas de estilo en cascada, nos sirven para poder darle un estilo a nuestras etiquetas de HTML. Como ya hemos dicho HTML y CSS van de la mano, si no usas CSS tendrás un sitio web muy estilo 90's sin color, sin posicionamientos, sin tamaños, sin dimenciones, etcétera.


Para dejarlo más claro, lo puedes tomar como tu cuerpo humano, el HTML es tu esqueleto y el CSS es tu piel o como tú te vez. Tú piel tiene un color, un grosor, un ancho y alto, un peso, tienes cabello o quizás no, etcétera. Básicamente es tu apariencia el CSS si lo tomamos en ese sentido.


¿Qué nos ofrece CSS3?

  • Animaciones y Transiciones.
  • Colores gradientes.
  • Rotación de elementos.
  • Uso de fuentes.
  • Pseudoclases
  • Genera sitios mas rápidos.
  • Utilización de Float, Flexbox y CSSgrid para un mejor manejo del HTML.
  • Manejo del margen exterior e interiror.
  • Diseño adaptivo.
  • Selector de etiqueta
  • Selectores (Class "." - ID "#").
